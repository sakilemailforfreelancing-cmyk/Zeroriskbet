# Phased rollout

## Phase 1 (MVP)

- Login and session flow with client API.
- Wallet balance and history.
- Lobby and one fully playable game with API-backed bets.
- PWA installability and offline shell.

## Phase 2

- Add 2-3 additional games using the shared game engine adapter contract.
- Improve game discovery: categories, search, featured cards.
- Add promotions and campaign placements if API exists.

## Phase 3

- Expand to full suite and advanced competition features.
- Add observability dashboards and richer analytics.
- Compliance hardening: responsible gaming and regional constraints.
